☁️ Synchronisation multi-appareils

Statut : Non configuré

Pour synchroniser tes données entre ton ordi et ton téléphone, connecte l'app à Firebase (service Google gratuit).

  1. Va sur console.firebase.google.com et connecte-toi avec ton compte Gmail.
  2. Clique sur "Créer un projet" → donne un nom (ex. yuki-cards) → continue → désactive Google Analytics si proposé → "Créer le projet".
  3. Dans le menu de gauche, clique sur "Realtime Database" → "Créer une base de données" → choisis l'emplacement → "Démarrer en mode test" → "Activer".
  4. Clique sur l'icône ⚙️ en haut à gauche → "Paramètres du projet" → descends jusqu'à "Vos applications" → clique sur l'icône </> → donne un nom (ex. yuki-web) → "Enregistrer l'application".
  5. Tu vois un bloc de code avec firebaseConfig. Copie les valeurs et colle-les dans les champs ci-dessous.
💡 Database URL — elle ressemble à :
https://TON-PROJET-default-rtdb.firebaseio.com
ou (si région Europe) :
https://TON-PROJET-default-rtdb.europe-west1.firebasedatabase.app

📱 Utiliser sur smartphone

Une fois Firebase configuré, voici comment accéder à l'app sur ton téléphone :

  1. Va sur app.netlify.com/drop (crée un compte gratuit si besoin).
  2. Glisse-dépose le dossier "app gestion collection" entier dans la zone prévue. Tu obtiens une URL du type https://nom-aléatoire.netlify.app
  3. Ouvre cette URL sur ton téléphone. Sur iPhone : bouton Partager → "Sur l'écran d'accueil". Sur Android : menu ⋮ → "Ajouter à l'écran d'accueil".
  4. L'app apparaît comme une vraie application sur ton écran d'accueil. Les données se synchronisent automatiquement via Firebase. ✅

⚠️ Après chaque mise à jour du fichier HTML, redépose-le sur GitHub pour que l'URL reste à jour.

🔄 Mises à jour de l'application

Vérifie si une nouvelle version est disponible. Si c'est le cas, un bandeau apparaîtra en bas de l'écran — clique sur "Recharger" pour l'installer sans perdre tes données ni ta connexion Firebase.

💾 Gestion des données

Exporte tes données en JSON (sauvegarde) ou en Excel (Google Sheets), ou importe un fichier existant.

💡 Une sauvegarde JSON est automatiquement téléchargée toutes les 24h.
